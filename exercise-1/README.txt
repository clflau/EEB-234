Adding, commiting, and pushing to github

commands involved:
When in directory that contains the file, make git track file by:
git add <filename>

To create new empty file with a given name:
touch <filename>

To open text file in gedit:
sudo gedit <filename>

To view status of git repository:
git status

To commit with a message:
git commit -m "write message here"

REMEMBER TO PUSH OR ELSE IT WON'T SHOW UP ON GITHUB!!

####
Branching

commands involved:
To check which branch I'm in:
git branch

To make new branch and go into the branch:
git checkout -b <branch name>

To see log of branches:
git log --decorate
